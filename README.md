# AURORA

Paperless Academic Defense Workflow System for Research, Capstone, Thesis & Dissertation Papers at Partido State University

## Tech Stack

- **Frontend:** Next.js 15, TypeScript, Tailwind CSS, shadcn/ui, React Query, Zustand
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Realtime, Edge Functions)
- **Deployment:** Vercel + GitHub

## Quick Start

```bash
# Install dependencies
npm install

# Configure environment
cp .env.local.example .env.local
# Edit .env.local with your Supabase credentials

# Run Supabase migrations (see docs/modules/MODULE-01-FOUNDATION.md)

# Start development server
npm run dev
```
